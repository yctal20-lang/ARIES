# Чекпоинт перед правками попапа и сидов

Состояние перед изменениями:

- Окно с данными о мусоре закрывается само через 6 секунд.
- Аномалии в симуляции срабатывают по условию `(step + seed) % 37 == 0` при step > 20.

Чтобы зафиксировать это состояние в git (если репозиторий инициализирован):

```bash
git add .
git commit -m "checkpoint: before popup close button and anomaly-free seeds"
```

## Сиды без аномалий

Для миссий без телеметрических аномалий используйте один из сидов:

### Примеры сидов

0, 1, 7, 8, 14, 15, 21, 22, 28, 29, 35, 36, 42, 43, 49, 50, 56, 57, 63, 64, 70, 71, 77, 78, 84, 85, 91, 92, 98, 99

Правило: сиды, у которых `seed % 7` равен **0** или **1**, не генерируют аномалий (при остальных сидах аномалии по-прежнему возможны).
